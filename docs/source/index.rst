OTM Documentation
=================

`OTM <https://github.com/ggomes/otm-sim>`_ is a traffic simulation platform that implements models and control algorithms as Java plugins. Three basic models are provided: the cell-transmission model, the two-queue (mesoscopic) model, and Newell's car-following model.
